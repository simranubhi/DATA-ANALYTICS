DATA FRAMES

A data frame is one of the most commonly used data structures in R. It is a two-dimensional, table-like structure where each column can contain different types of data (numeric, character, logical, etc.). It's similar to a spreadsheet or a SQL table and is widely used for storing and manipulating structured datasets.

Creating a Data Frame

You can create a data frame using the data.frame() function by combining vectors of the same length. Each vector becomes a column in the data frame. 

names <- c("Josh", "Jette", "James")
class <- c("9A", "10C", "12B")
rollno <- c(34, 56, 21)

student <- data.frame(Name = names, Class = class, Rollnumber = rollno)
print(student)

Accessing Specific Columns or Values

You can access a specific column in a data frame using the $ operator or by indexing with row and column numbers using [row, column]. 

print(student$Name)           # Access 'Name' column
print(student[3, 2])          # Access element at 3rd row, 2nd column

Adding a New Column

To add a new column, use the $ operator and assign a vector of values. The length must match the number of rows in the data frame. 

student$Marks <- c(78, 87, 34)
print(student)

Adding a New Row using rbind()

You can append a new row to a data frame using rbind(). The new row must be a data frame with the same structure. 

new <- data.frame(Name = "Laura", Class = "8C", Rollnumber = "23", Marks = "56")
student <- rbind(student, new)
print(student)

Updating Values Based on Conditions

Use logical indexing to modify values in the data frame based on specific conditions. 

student$Class[student$Name == "Jette"] <- "6D"
print(student)

Removing Rows Based on Conditions

To remove a row, filter the data frame using a condition. Only rows that meet the condition will be retained. 

student <- student[student$Name != "Josh", ]
print(student)

Deleting a Column

You can delete a column by assigning NULL to it using the $ operator. 

student$Marks <- NULL
print(student)

Sorting the Data Frame

Use the order() function to sort the data frame by a specific column. 

sorted <- student[order(student$Rollnumber), ]
print(sorted)

Column Names and Row Names

You can set or retrieve column and row names using the colnames() and rownames() functions. 

colnames(Data_Frame) <- c("Type", "HeartRate", "Time")
rownames(Data_Frame) <- c("Day1", "Day2", "Day3")

Changing Data Types

You can change the data type of columns using functions like as.character(), as.numeric(), and as.factor(). 

Data_Frame$Pulse <- as.character(Data_Frame$Pulse)
Data_Frame$Pulse <- as.numeric(Data_Frame$Pulse)

Using str(), head(), and tail()

- str(): Shows the structure of the data frame.
- head(): Displays the first 6 rows.
- tail(): Displays the last 6 rows. 

str(Data_Frame)
head(Data_Frame)
tail(Data_Frame)

TIBBLES

Tibbles are modern versions of data frames that come with several enhancements. They are part of the tidyverse and are designed to make data handling easier and safer.

Creating a Tibble

Tibbles are created using the tibble() function from the tibble package. Unlike data frames, they do not convert character variables to factors by default. 

library(tibble)
my_data <- tibble(
  name = c("Sandeep", "Amit", "Aman"),
  age = c(25, 30, 35),
  city = c("Pune", "Jaipur", "Delhi")
)
print(my_data)

Accessing Data in a Tibble

Tibbles support standard ways of accessing data: using $ for columns, [[ ]] for column by index, and [ , ] for row/column indexing. 

my_data$name
my_data[1, 2]
my_data[[3]]

Modifying a Tibble

Tibbles can be modified just like data frames. You can add or change columns by assignment. 

my_data$country <- c("India", "India", "India")
my_data$age[1] <- 26

Converting Data Frame to Tibble

You can convert a data frame to a tibble using as_tibble(). This is especially useful when you want the benefits of tidyverse compatibility. 

df <- data.frame(x = 1:3, y = c("a", "b", "c"))
tb <- as_tibble(df)
