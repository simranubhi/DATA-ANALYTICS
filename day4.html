DATA FRAMES

A data frame is a 2D table-like structure with columns of different types. It is widely used to store and manipulate structured datasets.

Creating a Data Frame

names <- c("Josh", "Jette", "James")
class <- c("9A", "10C", "12B")
rollno <- c(34, 56, 21)

student <- data.frame(Name = names, Class = class, Rollnumber = rollno)
print(student)

Accessing Specific Columns or Values

print("Names:")
print(student$Name)

print("Element at row 3 col 2")
print(student[3, 2])

Adding a New Column

student$Marks <- c(78, 87, 34)
print(student)

Adding a New Row using rbind()

new <- data.frame(Name = "Laura", Class = "8C", Rollnumber = "23", Marks = "56")
student <- rbind(student, new)
print(student)

Updating Values Based on Conditions

student$Class[student$Name == "Jette"] <- "6D"
print(student)

Removing Rows Based on Conditions

student <- student[student$Name != "Josh", ]
print(student)

Deleting a Column

student$Marks <- NULL
print(student)

Sorting the Data Frame

sorted <- student[order(student$Rollnumber), ]
print(sorted)

Column Names and Row Names

colnames(Data_Frame) <- c("Type", "HeartRate", "Time")
rownames(Data_Frame) <- c("Day1", "Day2", "Day3")

Changing Data Types

Data_Frame$Pulse <- as.character(Data_Frame$Pulse)
Data_Frame$Pulse <- as.numeric(Data_Frame$Pulse)

Using str(), head(), and tail()

str(Data_Frame)
head(Data_Frame)
tail(Data_Frame)

TIBBLES

Tibbles are modern, more consistent versions of data frames in R. Part of the tidyverse package, tibbles print cleaner and avoid automatic type conversion.

Creating a Tibble

library(tibble)
my_data <- tibble(
  name = c("Sandeep", "Amit", "Aman"),
  age = c(25, 30, 35),
  city = c("Pune", "Jaipur", "Delhi")
)
print(my_data)

Accessing Data in a Tibble

my_data$name
my_data[1, 2]
my_data[[3]]

Modifying a Tibble

my_data$country <- c("India", "India", "India")
my_data$age[1] <- 26

Converting Data Frame to Tibble

df <- data.frame(x = 1:3, y = c("a", "b", "c"))
tb <- as_tibble(df)
