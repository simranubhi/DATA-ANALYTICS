Data Structures in R

Data structures in R help organize data for analysis. They can be simple or complex and are tailored to specific data needs. Primary structures include vectors, lists, matrices, arrays, factors, and data frames.

1. Vectors

Vectors are the simplest structure. They hold elements of the same type and support vectorized operations (no loops needed).

numeric_vector <- c(10, 20, 30)
character_vector <- c("apple", "banana", "cherry")
logical_vector <- c(TRUE, FALSE, TRUE)

Accessing Elements:

vector[1]         # First element
vector[c(1, 3)]   # Multiple elements

Use Case: Storing marks, temperatures, or survey results.

2. Matrices

Matrices are 2D structures with elements of the same type. Used in linear algebra, image processing, and stats.

my_matrix <- matrix(1:9, nrow = 3, ncol = 3)
my_matrix[1,2]    # Element at row 1, column 2
my_matrix[2,]     # Entire 2nd row
my_matrix[,3]     # Entire 3rd column

Operations:

my_matrix + another_matrix
my_matrix * another_matrix
  
# Count total elements
length(hello)

# Nested for loop to print matrix elements
num <- matrix(c(10, 20, 30, 40), nrow = 2, ncol = 2)
for (rows in 1:nrow(num)) {
  for (columns in 1:ncol(num)) {
    print(num[rows, columns])
  }
}

# Sum of all elements
total_sum <- sum(num)
print(paste("Total sum:", total_sum))

3. Arrays

Arrays extend matrices to 3+ dimensions. Useful for simulations, image data, and environmental models.

my_array <- array(1:8, dim = c(2, 2, 2))

arr <- c(1:9)
thisarray <- c(1:10)
multiarray <- array(thisarray, dim = c(4, 3, 2))
multiarray[c(1), , 1]    # 1st row from 1st layer
dim(multiarray)          # Show dimensions

4. Lists

Lists are flexible and can hold mixed-type elements including other lists or data frames.

my_list <- list(name = "DataCamp", year = 2024, scores = c(80, 90, 85), active = TRUE)

my_list[[3]]    # Access by position
my_list$name    # Access by name

Use Case: Machine learning outputs, Shiny apps, nested reports.

5. Factors

Factors are categorical variables with a limited number of unique values (levels). Used in regression, grouping, and plotting.

gender <- factor(c("male", "female", "female", "male"))
levels(gender) <- c("Female", "Male")

# Summary stats
summary(gender)

Use Case: Gender, education level, product categories, etc.
