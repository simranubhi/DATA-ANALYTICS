Statistical Analysis

Statistical analysis involves collecting, organizing, summarizing, and interpreting data to uncover patterns and trends. It helps in making informed decisions by applying mathematical techniques such as:

  • Measures of Central Tendency: mean(), median()
  • Measures of Dispersion: var(), sd()
  • Measures of Position: quantile(), percentile()

Example:

myvector <- c(23, 43, 78, 21, 12, 32, 43, 54, 65, 16)

print(myvector)         # Displays the vector
sum(myvector)           # Sum of elements
min(myvector)           # Smallest value
max(myvector)           # Largest value
sort(myvector)          # Sorted vector
which.max(myvector)     # Index of max value
which.min(myvector)     # Index of min value
mean(myvector)          # Average
median(myvector)        # Middle value
var(myvector)           # Variance
quantile(myvector, 0.25) # 25th percentile
names(sort(-table(myvector))) # Mode workaround
mode(myvector)          # Data type (numeric)

Statistical Tests in R

We use statistical tests to draw inferences and evaluate hypotheses.

vect1 <- c(23, 34, 21, 12, 21, 34, 56, 62, 22, 18)
vect2 <- c(34, 21, 43, 23)

1. T-Test

Compares the means of two independent groups.

result = t.test(vect1, vect2)
print(result)

2. Chi-Square Test

Tests if observed frequencies differ from expected frequencies.

myChisq = vect1
chisq.test(myChisq)

3. F-Test

Checks if two samples have equal variances.

result = var.test(vect1, vect2)
print(result)

4. Correlation Test

Measures linear relationship between two numeric vectors.

cor_result = cor.test(vect1[1:4], vect2)
print(cor_result)

Graphs in R

Visualization helps in understanding data effectively. R provides a wide variety of charting tools.

Basic Plotting

plot(3, 2)
plot(c(2, 3), c(3, 2))
plot(1:10, col="magenta", type="s", xlab="TIME", ylab="DISTANCE", main="MY GRAPH")

Custom Symbols and Line Types

plot(1:10, cex=2, pch=0)  # Square symbol
plot(1:10, type="s", col="red", lwd=3, lty=6)  # Step chart with dashed lines

Multiple Line Charts

line1 <- c(1, 3, 5, 7)
line2 <- c(2, 4, 6, 8)
plot(line1, type="l", col="blue", lwd=3)
lines(line2, type="l", col="darkgreen", lwd=3)

Multiple Sets of Points

x1 <- c(1, 2, 3, 4)
y1 <- c(5, 6, 7, 8)
x2 <- c(2, 3, 4)
y2 <- c(7, 8, 9)
x3 <- c(3, 4, 5)
y3 <- c(5, 6, 7)

plot(x1, y1, col="red")
points(x2, y2, col="blue")
points(x3, y3, col="darkgreen")

Pie Charts

Pie charts show proportional data in circular form.

x <- c(100, 200, 300, 400, 200)
colors <- c("red", "darkgreen", "purple", "yellow", "orange")
mylabel <- c("apple", "pear", "jamun", "banana", "oranges")

pie(x, init.angle=100, mylabel, radius=0.75, main="MY PIE CHART", col=colors)
legend("bottomright", mylabel, fill=colors)

Bar Charts

Bar plots represent categorical data using vertical or horizontal bars.

x <- c("Housing", "Rent", "Electric", "Food", "Water")
y <- c(2, 4, 6, 8, 10)
colors <- c("red", "darkgreen", "purple", "yellow", "orange")

barplot(y, names.arg=x, col=colors, density=20, width=c(0.5), horiz=TRUE)
