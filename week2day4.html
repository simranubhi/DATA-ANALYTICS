    • Statistical Analysis

    Statistical analysis involves collecting, organizing, summarizing, and interpreting data to uncover patterns and trends. It helps in making informed decisions by applying mathematical techniques such as measures of central tendency (mean, median, mode), dispersion (variance, standard deviation), and position (quantiles, percentiles). In R, functions like mean(), median(), var(), and quantile() are commonly used for basic statistical analysis of numeric data.

    Example Code:

    myvector <- c(23, 43, 78, 21, 12, 32, 43, 54, 65, 16)

print(myvector)              # Displays contents of the vector
sum(myvector)                # Returns sum of all elements
min(myvector)                # Smallest value
max(myvector)                # Largest value
sort(myvector)               # Ascending order
which.max(myvector)          # Index of max value
which.min(myvector)          # Index of min value
mean(myvector)               # Average
median(myvector)             # Middle value
var(myvector)                # Variance
quantile(myvector, 0.25)     # 25th percentile
names(sort(-table(myvector))) # Most frequent values
mode(myvector)               # Internal data type (e.g., "numeric")

    Statistical Tests in R

    Input Vectors:

    vect1 <- c(23, 34, 21, 12, 21, 34, 56, 62, 22, 18)
vect2 <- c(34, 21, 43, 23)

    1. T-Test

    The T-Test compares the means of two independent samples to determine if there is a statistically significant difference between them.

    Purpose: To check whether the average values of two groups are significantly different.

    Syntax Used:

    result = t.test(vect1, vect2)
print(result)

    2. Chi-Square Test

    The Chi-Square Test is used to compare observed frequencies with expected frequencies, often applied to categorical data to test goodness-of-fit or independence.

    Purpose: To check if the distribution of categorical data differs from what we expect.

    Syntax Used:

    myChisq = vect1
chisq.test(myChisq)

    3. F-Test

    The F-Test is used to compare the variances of two samples to test if they come from populations with equal variances.

    Purpose: To verify if the spread (variance) of two datasets is significantly different.

    Syntax Used:

    result = var.test(vect1, vect2)
print(result)

    4. Correlation Test

    The Correlation Test measures the strength and direction of a linear relationship between two numeric variables.

    Purpose: To evaluate whether an increase in one variable is associated with an increase or decrease in another variable.

    Syntax Used:

    cor_result = cor.test(vect1[1:4], vect2)
print(cor_result)

    Multiple Line Charts

    Plots two lines on the same graph using plot() and lines() functions. Line colors and thickness are customized.

    line1 <- c(1, 3, 5, 7)
line2 <- c(2, 4, 6, 8)
plot(line1, type="l", col="blue", lwd=3)
lines(line2, type="l", col="darkgreen", lwd=3)

    Multiple Sets of Points

    Plots multiple groups of data points on a single graph using points(). Each group has a distinct color. 

    x1 <- c(1, 2, 3, 4)
y1 <- c(5, 6, 7, 8)
x2 <- c(2, 3, 4)
y2 <- c(7, 8, 9)
x3 <- c(3, 4, 5)
y3 <- c(5, 6, 7)
plot(x1, y1, col="red")
points(x2, y2, col="blue")
points(x3, y3, col="darkgreen")

    Pie Charts

    Pie chart is a circular chart divided into different segments according to the ratio of data provided. The total value of the pie is 100 and the segments tell the fraction of the whole pie. It is another method to represent statistical data in graphical form and pie() function is used to perform the same.

    Syntax:

    pie(x, labels, col, main, radius)
  • x is data vector
  • labels shows names given to slices
  • col fills the color in the slices as given parameter
  • main shows title name of the pie chart
  • radius indicates radius of the pie chart. It can be between -1 to +1

    • Example:

    x <- c(100, 200, 300, 400, 200)
colors <- c("red", "darkgreen", "purple", "yellow", "orange")
mylabel <- c("apple", "pear", "jamun", "banana", "oranges")
pie(x, init.angle=100, mylabel, radius=0.75, main="MY PIE CHART", col=colors)
legend("bottomright", mylabel, fill=colors)

    Bar Charts

    Bar plot or Bar Chart in R is used to represent the values in data vector as height of the bars. The data vector passed to the function is represented over y-axis of the graph. Bar chart can behave like histogram by using table() function instead of data vector.

    Syntax:

    barplot(data, xlab, ylab)
  • data is the data vector to be represented on y-axis
  • xlab is the label given to x-axis
  • ylab is the label given to y-axis

    • Example:

    x <- c("Housing", "Rent", "Electric", "Food", "Water")
y <- c(2, 4, 6, 8, 10)
colors <- c("red", "darkgreen", "purple", "yellow", "orange")
barplot(y, names.arg=x, col=colors, density=20, width=c(0.5), horiz=TRUE)
