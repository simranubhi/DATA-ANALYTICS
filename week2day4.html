Statistical Analysis

Statistical analysis involves collecting, organizing, summarizing, and interpreting data to uncover patterns and trends. It helps in making informed decisions by applying mathematical techniques such as:

  • Measures of central tendency (mean, median, mode)
  • Measures of dispersion (variance, standard deviation)
  • Measures of position (quantiles, percentiles)

In R, functions like mean(), median(), var(), and quantile() are commonly used for basic statistical analysis of numeric data.

Example

myvector <- c(23, 43, 78, 21, 12, 32, 43, 54, 65, 16)

print(myvector)        # Displays the contents of the vector
sum(myvector)          # Returns the sum of all elements
min(myvector)          # Smallest value
max(myvector)          # Largest value
sort(myvector)         # Ascending sort
which.max(myvector)    # Index of max value
which.min(myvector)    # Index of min value
mean(myvector)         # Average
median(myvector)       # Middle value
var(myvector)          # Variance
quantile(myvector, 0.25)  # 25th percentile
names(sort(-table(myvector)))  # Most frequent values
mode(myvector)         # Internal data type (e.g., "numeric")

Statistical Tests in R

To understand the application of statistical tests, consider two input vectors:

vect1 <- c(23, 34, 21, 12, 21, 34, 56, 62, 22, 18)
vect2 <- c(34, 21, 43, 23)

1. T-Test

Compares the means of two independent samples to determine if they are significantly different.

Purpose: To check whether the average values of two groups are significantly different.

result = t.test(vect1, vect2)
print(result)

2. Chi-Square Test

Used to compare observed frequencies with expected frequencies, often for categorical data.

Purpose: To check if the distribution of categorical data differs from what is expected.

myChisq = vect1
chisq.test(myChisq)

3. F-Test

Compares the variances of two samples to test if they come from populations with equal variances.

result = var.test(vect1, vect2)
print(result)

4. Correlation Test

Measures the strength and direction of a linear relationship between two numeric variables.

cor_result = cor.test(vect1[1:4], vect2)
print(cor_result)

Graphs in R

Graphs are essential for visualizing data in a clear and meaningful way. R provides several functions to create different types of plots and charts.

Basic Plotting

plot(3, 2)                              # Single point
plot(c(2, 3), c(3, 2))                  # Two points
plot(1:10, col="magenta", type="s", 
     xlab="TIME", ylab="DISTANCE", 
     main="MY GRAPH")                  # Step chart with labels

Custom Symbols and Line Types

plot(1:10, cex=2, pch=0)               # Square points
plot(1:10, type="s", col="red", lwd=3, lty=6)  # Red step chart with line style

Multiple Line Charts

line1 <- c(1, 3, 5, 7)
line2 <- c(2, 4, 6, 8)
plot(line1, type="l", col="blue", lwd=3)
lines(line2, type="l", col="darkgreen", lwd=3)

Multiple Sets of Points

x1 <- c(1, 2, 3, 4)
y1 <- c(5, 6, 7, 8)
x2 <- c(2, 3, 4)
y2 <- c(7, 8, 9)
x3 <- c(3, 4, 5)
y3 <- c(5, 6, 7)
plot(x1, y1, col="red")
points(x2, y2, col="blue")
points(x3, y3, col="darkgreen")

Pie Charts

Pie charts represent proportions of a whole in circular format using the pie() function.

Syntax: pie(x, labels, col, main, radius)

  • x: Data values
  • labels: Names for slices
  • col: Colors
  • main: Title
  • radius: Size of the pie
x <- c(100, 200, 300, 400, 200)
colors <- c("red", "darkgreen", "purple", "yellow", "orange")
mylabel <- c("apple", "pear", "jamun", "banana", "oranges")
pie(x, init.angle=100, mylabel, radius=0.75, main="MY PIE CHART", col=colors)
legend("bottomright", mylabel, fill=colors)

Bar Charts

Bar charts display values using rectangular bars. The height (or length) of the bar is proportional to the data value.

Syntax: barplot(data, xlab, ylab)

x <- c("Housing", "Rent", "Electric", "Food", "Water")
y <- c(2, 4, 6, 8, 10)
colors <- c("red", "darkgreen", "purple", "yellow", "orange")
barplot(y, names.arg=x, col=colors, density=20, width=c(0.5), horiz=TRUE)
