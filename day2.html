1. Variables in R

Variables store data values and are created using the <- or = operator.

x <- 10
y = "R Programming"

Variable names should start with a letter. <- is preferred in R programming.

2. Data Types in R

3. Operators in R

Arithmetic Operators

a + b  # addition
a - b  # subtraction
a * b  # multiplication
a / b  # division
a ^ b  # exponentiation
a %% b # modulo (remainder)
a %/% b # integer division

Relational Operators

a == b  # equal
a != b  # not equal
a > b   # greater than
a < b   # less than
a >= b  # greater than or equal
a <= b  # less than or equal

Logical Operators

&   # AND
|   # OR
!   # NOT

4. Vectors

A vector is a sequence of elements of the same type.

v1 <- c(1, 2, 3, 4, 5)
v2 <- c("a", "b", "c")
v3 <- c(TRUE, FALSE, TRUE)

5. Matrices

A matrix is a two-dimensional collection of elements.

m1 <- matrix(1:9, nrow = 3, ncol = 3)
m1[1, 2]  # first row, second column

6. Data Frames

A data frame holds data in tabular form.

df <- data.frame(
  ID = c(101, 102, 103),
  Name = c("Amit", "Sara", "John"),
  Score = c(90, 85, 88)
)
df$Name
